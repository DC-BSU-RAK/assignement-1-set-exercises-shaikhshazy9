HTML & CSS Quick Reference Guide

This guide covers essential HTML tags and CSS properties that every web developer should know. Each section includes a description and example code.

HTML Tags

HTML (HyperText Markup Language) is the standard language for creating web pages.

Headings (h1-h6)

Headings define the structure of your document. <h1> is the most important heading, while <h6> is the least important.

<h1>This is a Main Heading</h1>
<h2>This is a Secondary Heading</h2>
<h3>This is a Tertiary Heading</h3>
<h4>This is a Fourth Level Heading</h4>
<h5>This is a Fifth Level Heading</h5>
<h6>This is a Sixth Level Heading</h6>

Paragraphs (p)

The paragraph tag creates a block of text separated from surrounding content by spaces.

<p>This is a paragraph of text. Paragraphs are block-level elements that automatically add space before and after the content.</p>

Links (a)

The anchor tag creates hyperlinks to other web pages or to sections within the current page.

<a href="https://www.example.com">Visit Example Website</a>
<a href="#html-section">Jump to HTML Section</a>

Lists (ul, ol, li)

HTML provides tags for creating ordered lists (numbered) and unordered lists (bulleted).

<!-- Unordered list -->
<ul>
    <li>First bullet item</li>
    <li>Second bullet item</li>
</ul>

<!-- Ordered list -->
<ol>
    <li>First numbered item</li>
    <li>Second numbered item</li>
</ol>

Images (img)

The image tag embeds images into your webpage. The src attribute specifies the path to the image, and alt provides alternative text.

<img src="image.jpg" alt="Description of the image">

Divisions (div)

The div tag defines a division or section in an HTML document and is used as a container for other HTML elements.

<div class="container">
    <h2>This is inside a div</h2>
    <p>This paragraph is also inside the same div.</p>
</div>

Spans (span)

The span tag is an inline container used to mark up a part of a text or document.

<p>Here is some text with a <span style="color: red;">colored span</span> in the middle.</p>

Tables (table, tr, th, td)

HTML tables allow web developers to arrange data into rows and columns.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>

Forms (form, input, button)

HTML forms are used to collect user input. Form elements include text fields, checkboxes, radio buttons, and more.

<form action="/submit-form" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Submit</button>
</form>

Code and Pre (code, pre)

The code tag defines a piece of computer code. The pre tag preserves both spaces and line breaks, making it ideal for code examples.

<pre>
<code>
function greet() {
    console.log("Hello, world!");
}
</code>
</pre>

Section (section)

The section tag defines a section in a document, like chapters, headers, footers, or any other sections of the document.

<section>
    <h2>Section Title</h2>
    <p>This is content inside a section.</p>
</section>

CSS Properties

CSS (Cascading Style Sheets) is used to style and layout web pages.

Color

The color property sets the color of text.

p {
    color: blue;
}

h1 {
    color: #FF5733; /* Hexadecimal */
}

span {
    color: rgb(255, 0, 0); /* RGB */
}

Background

The background properties set the background color or image of an element.

body {
    background-color: #f0f0f0;
}

header {
    background-image: url('background.jpg');
    background-repeat: no-repeat;
    background-size: cover;
}

Font Properties

Font properties control the appearance of text, including the font family, size, weight, and style.

p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
}

Margin and Padding

Margin creates space outside an element, while padding creates space inside an element.

div {
    margin: 10px; /* Space outside the div */
    padding: 20px; /* Space inside the div */
}

/* Individual sides */
section {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 10px;
    margin-left: 15px;
    
    padding-top: 5px;
    padding-right: 10px;
    padding-bottom: 5px;
    padding-left: 10px;
}

Border

The border property adds a border around an element.

div {
    border: 1px solid black;
}

/* Individual properties */
button {
    border-width: 2px;
    border-style: dashed;
    border-color: red;
    border-radius: 5px; /* Rounded corners */
}

Display

The display property specifies how an element is displayed.

div {
    display: block; /* Default for div */
}

span {
    display: inline; /* Default for span */
}

nav li {
    display: inline-block; /* Combines properties of both */
}

.hidden {
    display: none; /* Hides the element */
}

Position

The position property specifies the type of positioning method used for an element.

div {
    position: static; /* Default */
}

header {
    position: fixed; /* Fixed to the viewport */
    top: 0;
    left: 0;
    width: 100%;
}

.relative-parent {
    position: relative; /* Establishes positioning context */
}

.absolute-child {
    position: absolute; /* Positioned relative to nearest positioned ancestor */
    top: 10px;
    right: 10px;
}

Text Properties

Text properties control the alignment, decoration, and spacing of text.

p {
    text-align: center; /* Alignment: left, right, center, justify */
    text-decoration: underline; /* Decoration: underline, overline, line-through */
    text-transform: uppercase; /* Capitalization: lowercase, uppercase, capitalize */
    line-height: 1.5; /* Space between lines */
    letter-spacing: 2px; /* Space between characters */
}

Flexbox

Flexbox is a layout model that allows responsive elements within a container.

.container {
    display: flex; /* Enables flexbox */
    flex-direction: row; /* Direction: row, column, row-reverse, column-reverse */
    justify-content: space-between; /* Horizontal alignment */
    align-items: center; /* Vertical alignment */
    flex-wrap: wrap; /* Allows items to wrap to next line */
}

.item {
    flex: 1; /* Grow and shrink equally */
}

.important-item {
    flex: 2; /* Takes twice as much space as other items */
}
